How to run 

NOTE: Make sure node Js and python3 installed on your system

1) First go to ReactCLient Folder and run this command : npm i 
2) For run reactClient run this command in reactClient directry: npm start

React client should start on port 3000

3) Now go to python directry and run this command : pip install -r requirements. txt
4) Now run this command in python directry to run flask: python3 app.py

Flask backend should run on port 8000

now upload file from reactClient and click on send to uplaod.
